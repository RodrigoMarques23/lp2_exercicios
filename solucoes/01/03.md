## Pergunta

3 - Indica o que é mostrado no ecrã pelo seguinte código C#. Justifica a tua resposta.

```cs
 byte b = byte.MaxValue - 1;

for (int i = 0; i < 5; i++)
{
    Console.WriteLine($"Iteração {i}: {(byte) (++b - 1)}");
}
```

## Soluções

### Solução 1

É impresso o seguinte:
```cs
Interação: 254
Interação: 255
Interação: 0
Interação: 1
Interação: 
```
Isto porque o máximo valor do byte é 255, logo o byte b será 254.
**No ciclo for é incrementado ```++b``` onde b é 255, mas o que é impresso é o valor de b - 1, sendo assim impresso 254.
De seguida b passa a ser 0 (pois dá a volta) sendo impresso 255 pois é retirado 1.
Assim sucessivamente enquanto ```i<5```, quando chegar a cinco, já nada é impresso.
